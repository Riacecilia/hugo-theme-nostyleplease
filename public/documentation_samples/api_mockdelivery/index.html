REST API documentation for a mock delivery company

Project Details

Goal

Document the mock delivery API using the Open API specification.

Company Background

It is a fictional company called Mock Delivery. It operates in the logistics industry. It manages deliveries of parcels worldwide. The API’s functionality keeps track of: deliveries, drivers, delivery hubs, delivery assignments and reporting.

The mock API was created by Wisdom Nwokocha, lead mentor of the Technical Writing Mentorship Program.

Deliverables

  • test the API and make sure all endpoints are working as expected
  • Produce reference documentation for the API using the Open API Specification (OAS) 3.0
  • host the complete documentation on Gitbook

Accomplishments

Technology Stack

  • API testing: Postman
  • OAS validator: Swagger UI
  • Text editor: Visual Studio Code
  • Markup languages: yaml, json, markdown
  • Static Site Generator: Gitbook
  • json to openapi converter: https://kevinswiber.github.io/postman2openapi/?ref=hackernoon.com
