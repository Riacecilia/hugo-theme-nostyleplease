## Problem

A delivery company has just launched their REST API. They need documentation that can be used by various stakeholders.


---

## Solution
- I tested the API using Postman to make sure all endpoints were working as expected.
- Created API reference documentation using the Open API Specification (OAS) 3.0
- Hosted the complete API reference and concept documentation on Gitbook.
